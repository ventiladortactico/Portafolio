SubeSude

Desarrollé SubeSude, una plataforma de video para adultos con flujo completo: registro con verificación por correo y OAuth de Google, roles y solicitud de creador, subida y gestión de contenido, interacciones sociales (likes, favoritos, seguimiento), analíticas para creadores y gating Premium. Implementé seguridad con sesiones, CSRF y reCAPTCHA, consultas preparadas y validación de archivos. El frontend usa PHP templating, HTML/CSS y JS nativo con UI rápida, paginación y lazy-loading; el backend en PHP con PDO/MySQL, PHPMailer para emails y dotenv para configuración. El resultado es una base sólida, extensible y lista para integrar pagos y más métricas.

Tecnologías

  • Backend/BD: PHP (PDO), MySQL
  • Frontend: HTML5, CSS3, JavaScript (Vanilla)
  • Email/OAuth: PHPMailer, Google API Client (OAuth)
  • Utilidades: vlucas/phpdotenv, Font Awesome
  • Seguridad: reCAPTCHA v2
  • Entorno: WAMP/Apache (estructura PHP)

Características

  • Autenticación: email/contraseña y OAuth de Google
  • Verificación y anti‑bot: verificación por correo y reCAPTCHA
  • Roles y acceso: usuario, creador, admin
  • Creador: solicitud y aprobación desde panel
  • Subida de medios: videos, miniaturas y avatares con validación de tipo/tamaño
  • Interacciones: likes/dislikes, favoritos, seguimiento a creadores
  • Historial: visualización y guardados
  • Búsqueda y UX: búsqueda unificada (categorías y creadores) y paginación
  • Premium: gating de contenido para suscriptores
  • Analíticas: vistas, likes, seguidores, top videos
  • UI: responsive con notificaciones, lazy‑loading y modales
  • Gestión: categorías y tendencias (últimos 30 días)
