Empleando

Empleando es una plataforma que conecta clientes con profesionales mediante un flujo simple: los clientes publican solicitudes y los trabajadores se postulan según sus categorías. Implementé el backend en PHP con MySQL utilizando consultas preparadas y sesiones para gestionar la autenticación, y diseñé una interfaz moderna con Bootstrap enfocada en claridad y rapidez. Integré subida y validación de imágenes, perfiles con calificación y un panel de administración para mantener la calidad del contenido. Priorizo la seguridad básica (validación de entradas, manejo de archivos y uso de sentencias preparadas) y una experiencia de usuario fluida con búsquedas y modales. El resultado es un sistema modular y mantenible que resuelve el emparejamiento entre demanda y talento de forma eficiente.

Tecnologías

  • Backend: PHP (mysqli), sesiones, respuestas JSON
  • Base de datos: MySQL
  • Frontend: HTML5, CSS3, Bootstrap 5, JavaScript
  • Email: PHPMailer
  • Paquetería: Composer
  • Entorno: WAMP/Apache
  • Otros: Subida/validación de imágenes, consultas preparadas

Características

  • Autenticación: registro/inicio de sesión con sesiones
  • Perfiles: cliente y trabajador con edición de dirección, descripción e imagen
  • Matching: exploración de trabajos por categorías del trabajador
  • Publicación: creación y gestión de solicitudes de servicio
  • Postulaciones: envío con control de duplicados e imagen opcional
  • Reseñas: calificaciones y comentarios a trabajadores
  • Admin: panel para usuarios, solicitudes, postulaciones, profesiones y reseñas
  • UI/UX: diseño dark con Bootstrap, modales y animaciones
